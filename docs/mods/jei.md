# JEI - Just Enough Items

View crafting recipes for any item in the game.

## How to Use

### View a Recipe

1. Hover over any item (in inventory, chest, or hotbar)
2. Press **R**
3. See how to craft it!

### View Uses

1. Hover over any item
2. Press **U**
3. See what you can make with it

### Search Items

1. Open any inventory (your own, a chest, crafting table)
2. Click the search bar on the right side
3. Type to filter items

## Search Tips

| Search | Finds |
|--------|-------|
| `iron` | All items with "iron" in the name |
| `@minecraft` | Only vanilla items (not modded) |
| `#forge:ores` | Items tagged as ores |
| `$pickaxe` | Items in the pickaxe creative tab |

## Common Uses

- **Forgot a recipe** - What's the pattern for a piston again? Press R
- **New items** - Found something weird? Check what it crafts into
- **Planning** - See all ingredients needed before you start

!!! note "Server-Side"
    JEI on this server is **server-side** - it works without installing anything, but may not show all features a client-side JEI would have.
