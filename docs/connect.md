# How to Connect

## :material-minecraft: Java Edition

All servers use SRV records - just enter the address, no port needed!

1. Open Minecraft Java Edition
2. Click **Multiplayer**
3. Click **Add Server**
4. Enter the server address (see table below)
5. Click **Done**, then double-click to join

## :material-cellphone: Bedrock Edition
