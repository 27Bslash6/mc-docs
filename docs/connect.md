# How to Connect

## Java Edition

1. Open Minecraft Java Edition
2. Click **Multiplayer**
3. Click **Add Server**
4. Enter the server address (see individual server pages)
5. Click **Done**, then double-click the server to join

## Bedrock Edition (idk server only)

The **idk** server supports Bedrock Edition (Xbox, PlayStation, Switch, Mobile, Windows 10/11).

1. Open Minecraft Bedrock Edition
2. Go to **Play** â†’ **Servers**
3. Scroll down and click **Add Server**
4. Enter:
    - **Server Name**: idk
    - **Server Address**: `mc.fnord.lol`
    - **Port**: `19132`
5. Click **Save** and join!

!!! warning "Bedrock Only Works on idk"
    Only the **idk** server supports Bedrock. The modpack servers (Pixelmon, Quiet Place, Zombie Invade) require Java Edition with mods installed.
