# Zombie Invade Server

Survive 100 days of zombie apocalypse chaos.

| **Difficulty** | Hard |

## What is This Modpack?

An apocalyptic survival challenge:

- **Zombie hordes** - Massive waves of undead
- **Day/night cycle matters** - Nights are deadly
- **Progression system** - Unlock better gear as you survive
- **Base building** - Fortify against the horde
- **100 day goal** - Can you survive?

## Installing the Modpack

### Prism Launcher (Recommended)

1. Download [Prism Launcher](https://prismlauncher.org/download/)
2. Click **Add Instance** → **Modrinth** tab
3. Search for the modpack name (ask Dad if unsure)
4. Select the **1.20.1** version
5. Click **OK** to install
6. Launch and connect!

!!! tip "Can't Find It?"
    Some modpacks are on CurseForge only. In Prism Launcher, try the **CurseForge** tab if Modrinth doesn't have it.

## Connecting

1. Launch the modpack from Prism Launcher
2. Go to **Multiplayer** → **Add Server**
3. Enter: `zombie.fnord.lol`
4. Survive!

## Survival Tips

- **Find shelter before night** - First night is critical
- **Fortify entrances** - Zombies will break through weak blocks
- **Torches everywhere** - Light prevents spawns
- **Team up** - Survival is easier together
- **Don't hoard** - If you die, your stuff is gone

!!! tip "First Night"
    Dig into a hillside and seal yourself in if you can't build a proper shelter in time.
