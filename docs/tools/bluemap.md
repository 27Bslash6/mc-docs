# BlueMap - World Map

Explore the idk server world in 3D at **[map.fnord.lol](https://map.fnord.lol)**

## Features

- **3D terrain** - See mountains, caves, and builds from any angle
- **Live players** - See who's online and where they are
- **Zoom levels** - From world overview to individual blocks
- **Day/night toggle** - See how the world looks at different times

## Controls

| Action | How |
|--------|-----|
| **Rotate** | Click and drag |
| **Zoom** | Scroll wheel |
| **Pan** | Right-click and drag |
| **Player list** | Click the icon in the corner |

## Uses

### Finding Your Death Location

1. Open the map
2. Look for your gravestone
3. Note the coordinates or plan a route

### Planning Builds

1. Find a good location on the map
2. Check the terrain and nearby builds
3. Get coordinates to navigate in-game

### Exploring

1. See what's been discovered
2. Find unexplored areas
3. Plan expeditions

## Map Updates

The map updates automatically as players explore. New chunks appear after being loaded in-game.

!!! note "Privacy"
    Player locations are visible on the map. If you want to keep your base secret, consider building underground or in an unexplored area.
