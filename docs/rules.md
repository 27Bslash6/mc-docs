# Server Rules

## The Golden Rules

1. **Be kind** - We're all here to have fun
2. **No griefing** - Don't destroy other people's builds
3. **No stealing** - Ask before taking from chests
4. **Respect claims** - Use FLAN to protect your stuff on idk
5. **No cheating** - No hacks, x-ray, or exploits

## Building Rules

- **Build at least 100 blocks away** from other players' bases (unless invited)
- **Clean up your mess** - Remove floating trees and creeper holes
- **Don't build inappropriate things** - Keep it family-friendly

## Chat Rules

- No swearing
- No spamming
- Be helpful to new players

## Consequences

Breaking rules may result in:

1. **Warning** - First offense
2. **Temporary ban** - Repeat offenses
3. **Permanent ban** - Serious violations

!!! tip "When in Doubt"
    If you're not sure if something is allowed, ask first!
